## Features

- Choose difficulty levels: Easy, Medium, or Hard.
- Limited chances based on difficulty:
  - Easy: 10 attempts
  - Medium: 5 attempts
  - Hard: 3 attempts
- Tracks the **fewest attempts** and stores the **all-time high score** in a JSON file.
- Displays time taken to guess the number.
- Clear terminal interface for better experience.

---

## Game Rules

1. Select a difficulty level.
2. Guess the number between 1 and 100.
3. The game will give hints: "Higher..." or "Lower..."
4. Try to guess the number in the fewest attempts possible.
5. If you beat the high score, your name and score are saved as the new record.
